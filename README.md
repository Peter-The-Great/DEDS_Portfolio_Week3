## Install
Om dit project te installeren moet je het volgende doen:
1. Clone het project
2. Open het project in je favoriete code editor
3. Zorg dat je poetry hebt geinstalleerd:
    ```bash
    pip install poetry
    of
    pipx install poetry
    ```
4. Installeer de dependencies:
    ```bash
    poetry install
    ```
5. Ga maar los.
